# Sistema de Perfil Universitario — Symfony Edition

Proyecto académico desarrollado en Symfony 7.x para la materia de Programación Avanzada.

## Arquitectura Técnica
- **Motor de plantillas:** Twig.
- **ORM:** Doctrine para persistencia de datos.
- **Patrón:** Modelo-Vista-Controlador (MVC) con inyección de dependencias.

## Requisitos de Ejecución
1. PHP 8.2+
2. Composer
3. Servidor MySQL (XAMPP)
